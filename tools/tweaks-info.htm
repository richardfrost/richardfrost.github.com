Applying Registry Files

To add one of these registry files to your system follow these steps:

1.       Right-click the link for the desired registry tweak (see below for options) and choose save link.

2.       Once the file has been downloaded, double-click on the .reg file and answer Yes or Allow to the prompts.

a.       If the file will not run you may need to Unblock it by right-clicking the file and choosing properties.

Standard (Recommended)

Context Menu

·         Copy TXT file contents to clipboard. (All Users)

·         Open with Notepad for all file types. (All Users)

·         Run VBS files as Administrator. (All Users)

·         Desktop Menu: Run dialog & Power Menu (Shutdown, Restart, etc.). (All Users)

Expanded Context Menu (Shift + Right-click)

·         Open Command Window Here as Administrator. (All Users)

·         Take Ownership of files and folders (Advanced). (All Users)

Other Settings

·         Add several useful commands into the "Run" history described below.

·         Disable Aero Shake.

·         Disable hiding known file extensions (Show all file extensions).

·         Disable "Internet Open With" dialog for unknown file types.

·         Disable "Shortcut to" from being prepended to shortcuts.

·         Disable Windows Update auto-reboot. (All Users)

·         Prevent Windows Update from hijacking the shutdown button.

Expanded

Same features as standard, but places entries from the Expanded Context Menu to the standard Menu.

Undo

Removes entries added by tweaks-standard.reg and tweaks-expanded.reg and restores defaults.
